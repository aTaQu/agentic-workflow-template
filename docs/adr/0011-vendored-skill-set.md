# Vendored skill set: the seven, plus architecture & diagnose

The repo vendors nine skills from `github.com/mattpocock/skills`: `tdd`, `grill-with-docs` (with
its `ADR-FORMAT.md` / `CONTEXT-FORMAT.md` support files), `grill-me`, `handoff`, `prototype`,
`teach`, and `parallel-batch` (the seven requested), plus `improve-codebase-architecture` and
`diagnose` — both of which read `CONTEXT.md` and `docs/adr/` and complement the loop's build/debug
phases. Skipped `setup-matt-pocock-skills` and the issue-tracker skills (`to-issues` / `triage` /
`to-prd`): none of the chosen skills require the `docs/agents/` issue-tracker scaffold, so it would
be unused. All are easy to add later from the same upstream repo.
