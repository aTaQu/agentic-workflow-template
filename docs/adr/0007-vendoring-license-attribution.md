# Public repo; vendored MP skills (MIT) retained with attribution

The repo is public under the personal `aTaQu` account. It vendors several of Matt Pocock's skills,
which are MIT-licensed; each vendored skill directory retains its upstream MIT license and
attribution, and the repository carries its own MIT `LICENSE`. Skills are vendored **fresh from
github.com/mattpocock/skills**, not copied from the local machine — the local clone is stale (it
was missing `teach` and may lag upstream in other ways).
